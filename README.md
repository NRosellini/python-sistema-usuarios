# Sistema de Login y Gestión de Alumnos

Proyecto integrador del curso de Python en Educación IT. Simula un sistema básico de autenticación con usuario y contraseña, y permite gestionar una lista de alumnos con la cantidad de cursos que está tomando cada uno.

## Funcionalidades

- Validación de usuario y contraseña al ingresar
- Menú interactivo con 3 opciones:
  - Añadir un alumno con su cantidad de cursos
  - Listar los alumnos cargados
  - Salir del sistema
- Manejo de opciones inválidas

## Conceptos aplicados

- Variables y tipos de datos
- Listas y diccionarios
- Entrada de datos con `input()`
- Estructuras condicionales (`if`, `elif`, `else`)
- Bucle `while`
- Manejo de menú con múltiples opciones

## Cómo ejecutarlo

python proyecto_integrador.py
